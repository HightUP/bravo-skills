# Criativos — [Cliente]

Um arquivo por criativo. Nome no formato:

```
TIPO-CONCEITO-VERSAO.md
```

Ex: `carrossel-look-completo-v3.md`, `video-prova-social-15s.md`

## Estrutura de cada nota

- **Tipo** (carrossel, single, vídeo)
- **Conceito**
- **Status** (em produção, ativo, pausado, arquivado)
- **Texto do anúncio**
- **Onde está a arte** (caminho/link Drive)
- **Onde rodou** (links pras campanhas)
- **Resultados consolidados**
- **Aprendizados**
